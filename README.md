# distributed-parallel-algorithms-labs
Distributed &amp; Parallel Algorithms Lab Solutions - UNSTPB FILS II-Eng - Year IV Sem II

## Clone With Submodules

Use this command to clone the repository and fetch submodules in one step:

```bash
git clone --recurse-submodules git@github.com:mariosakka/distributed-parallel-algorithms-labs.git
```

If you already cloned without submodules, run:

```bash
git submodule update --init --recursive
```
